What is SHAP?

SHAP (SHapley Additive exPlanations) explains how each feature contributes to a model's prediction. The visualizations below show which transaction characteristics most influence fraud detection.

  • Bar Chart - Shows the average impact of each feature on the prediction
  • Summary Plot - Shows how each feature affects predictions (high/low values)
  • Red/Blue colors indicate whether higher feature values increase or decrease fraud risk
